# ws-spring-boot-starter

### 描述

扩展可选功能

pom.xml添加相关依赖、仓库坐标

``` xml

<repositories>
    <repository>
        <id>github</id>
        <name>GitHub OWNER Apache Maven Packages</name>
        <url>https://maven.pkg.github.com/WangShuHunYin/ws-spring-boot-starter</url>
    </repository>
    <repository>
        <id>github</id>
        <name>GitHub OWNER Apache Maven Packages</name>
        <url>https://maven.pkg.github.com/WangShuHunYin/ws-base</url>
    </repository>
</repositories>

<dependencies>
    <dependency>
        <groupId>com.ws</groupId>
        <artifactId>ws-base</artifactId>
        <version>1.0.2</version>
    </dependency>

    <dependency>
        <groupId>com.ws</groupId>
        <artifactId>ws-spring-boot-starter</artifactId>
        <version>1.0.2</version>
    </dependency>
</dependencies>

```
